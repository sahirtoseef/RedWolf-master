CVSS Score

This dashboard shows valued CVSS scores of each CVE. The size of each circle corresponds to its associated number of occurences.


You may use the hover function to see each element of the dashboard and get more details. You can also use the tableau features in the bottom right to get the link to share, download the data, or see the data in fullscreen.
